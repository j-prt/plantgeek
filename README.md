# plantgeek

## an app for houseplant lovers


### Features
#### Plants
- [x] view all plants in database (MongoDB)
- [x] filter plants by type
- [x] search plants by name or type
- [x] view plant care info for each plant

#### Users
- [x] create an account (users saved to MongoDB with hashed passwords)
- [x] log in and out (remain logged in for 24 hours)
- [x] save plants to your wishlist
- [x] save your favorite plants
- [x] show off your collection of plants
- [x] view your plants on your profile
- [x] add & remove friends
- [x] upload a profile picture
- [x] comment on plants
