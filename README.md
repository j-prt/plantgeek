# plantgeek

### an app for houseplant lovers
- [Video Tour](https://drive.google.com/file/d/1dgOxPwHq7GW33dQjWKoTZAOVQHgwteT_/view?usp=sharing)

## Feature Summary
### Plants
- [x] view all plants in database (MongoDB)
- [x] filter plants by type
- [x] search plants by name or type
- [x] view plant care info for each plant

### Users
- [x] create an account (account saved to MongoDB with hashed password)
- [x] log in and out
- [x] show off your personal collection
- [x] keep a list of favorite plants
- [x] create a wishlist
- [x] make friends
- [x] upload a profile picture
- [x] comment on plants

### Style
- [x] responsive (looks good in desktop, tablet, & mobile mode)
