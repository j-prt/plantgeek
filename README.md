# plantgeek

### Disclaimer
This app started as a school project in 2021 and has since undergone major changes and improvements, particularly in the backend. It is currently an ongoing side project I work on in my free time.

### an app for houseplant lovers
- [Video Tour](https://youtu.be/_LXWqhxIMrQ)

## Feature Summary
### Plants
- [x] search & filter a database of hundreds of houseplants
- [x] view detailed plant care info for each plant

### Users
- [x] create an account
- [x] manage your collection, favorites, and wishlist
- [x] upload a profile picture

### Style
- [x] responsive across mobile, tablet, and desktop
